# comicz-cli 说明文档
一个下载漫画的命令行工具。

## 安装
```
npm install comicz-cli -g
```

## 使用方法
1. 终端输入
```
comicz-cli
```
2. 选择漫画收录来源
![](doc/1.png)
3. 检索查找的漫画名称
![](./doc/2.png)
4. 选择需要下载的结果
![](./doc/3.png)
5. 选择需要下载的章节（操作方式：空格选择，a选择所有，i反选，Enter确定
![](./doc/4.png)
6. 等待下载完成
![](./doc/5.png)
